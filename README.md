A modern and responsive weather application built with **React** that provides real-time weather data, a 7-day forecast, and hourly weather updates using the **Open-Meteo API**.

![Preview](./screenshot.jpg)

## ✨ Features

- 🔍 Search weather by city name
- 📍 Location suggestions using Geocoding API
- 🌡️ Current weather conditions
- 💧 Humidity, Wind Speed & Precipitation
- 📅 7-Day Forecast
- 🕒 Hourly Forecast for the selected day
- 🌤️ Dynamic weather icons
- 🌍 Metric & Imperial unit conversion
- 📱 Fully responsive design
- ⚠️ API error and "No Results" handling

## 🛠️ Built With

- React
- React Context API
- Axios
- CSS3
- Vite
- Open-Meteo Forecast API
- Open-Meteo Geocoding API

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/weather-app.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
